# Ingestion Scripts

This directory is reserved for content acquisition and normalization tooling.

Planned responsibilities:

- source fetchers for structured files, web pages, and PDFs
- parser adapters for source-specific cleanup
- normalization helpers that emit stable internal records
- validation utilities for provenance, metadata completeness, and verse-link targets

No production ingestion scripts are implemented yet. Early app phases use seeded content shaped to match the intended normalized model.
